## 🤖 AI 분류 로직

챗봇이 메시지를 분석해 관련 탭으로 안내합니다.

```json
{
  "intent": "hospital",
  "recommended_module": "hospital",
  "urgency_level": "high",
  "requires_hospital": true,
  "location_required": true
}
```

| 키워드 예시 | 연결 탭 | 긴급도 |
|------------|---------|--------|
| 구토, 혈뇨, 설사, 기침 | 🏥 병원 | 🔴 high |
| 사료, 모래, 직구, 쇼핑 | 🛒 쇼핑 | 🟢 low |
| 합사, 입질, 행동 | 📚 교육 | 🟢 low |
| 동네, 캣시터, 길냥이 | 🏘️ 커뮤니티 | 🟢 low |

## 🚀 실행 방법

### 프론트엔드
```bash
cd web
npm install
npm run dev
# → http://localhost:5173
```

### 환경변수 설정 (선택)
```bash
# web/.env.local 파일 생성 (git에 포함되지 않음)
VITE_OPENAI_API_KEY=sk-...   # GPT 챗봇 활성화
```

> ⚠️ `.env.local`은 `.gitignore`에 의해 자동으로 제외됩니다. 절대 커밋하지 마세요.

### 백엔드 (선택)
```bash
pip install -r requirements.txt
uvicorn main:app --reload --port 8001
# → http://localhost:8001/docs
```

### Vercel 배포
```bash
cd web
npx vercel --prod
```
